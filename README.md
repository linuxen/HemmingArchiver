# HamArc

HamArc — это консольный архиватор файлов.
Программа создаёт архивы собственного формата `.haf`, хранит в них файлы и использует код Хэмминга для защиты данных от одиночных битовых ошибок.

## Возможности

Архиватор поддерживает следующие операции:

* создание нового архива;
* просмотр списка файлов в архиве;
* извлечение всех файлов из архива;
* извлечение отдельных файлов из архива;
* добавление файлов в существующий архив;
* удаление файлов из архива;
* объединение двух архивов в один;
* кодирование и декодирование данных с помощью кода Хэмминга.

## Сборка

Для сборки проекта используется CMake.

```bash
cmake -S . -B build
cmake --build build
```

После сборки исполняемый файл находится по пути:

```bash
./build/bin/hamarc
```

## Использование

Общий формат команды:

```bash
./build/bin/hamarc <команда> -f <архив> [файлы]
```

Где:

* `<команда>` — действие, которое нужно выполнить;
* `<архив>` — имя архива;
* `[файлы]` — список файлов, если команда требует их указания.

## Команды

| Команда | Полная форма    | Описание                            |
| ------- | --------------- | ----------------------------------- |
| `-c`    | `--create`      | создать новый архив                 |
| `-l`    | `--list`        | вывести список файлов в архиве      |
| `-x`    | `--extract`     | извлечь файлы из архива             |
| `-a`    | `--append`      | добавить файлы в существующий архив |
| `-d`    | `--delete`      | удалить файлы из архива             |
| `-A`    | `--concatenate` | объединить два архива               |
| `-f`    | `--file`        | указать имя архива                  |

## Создание архива

Чтобы создать архив, используется команда `-c` или `--create`.

```bash
./build/bin/hamarc -c -f archive.haf file1.txt file2.txt
```

То же самое через полные имена команд:

```bash
./build/bin/hamarc --create --file archive.haf file1.txt file2.txt
```

После выполнения команды будет создан архив `archive.haf`, содержащий файлы `file1.txt` и `file2.txt`.

## Просмотр содержимого архива

Чтобы посмотреть список файлов внутри архива, используется команда `-l` или `--list`.

```bash
./build/bin/hamarc -l -f archive.haf
```

Пример вывода:

```text
file1.txt 128 bytes
file2.txt 256 bytes
```

## Извлечение всех файлов

Чтобы извлечь все файлы из архива, используется команда `-x` или `--extract`.

```bash
./build/bin/hamarc -x -f archive.haf
```

Все файлы из архива будут извлечены в текущую директорию.

## Извлечение отдельных файлов

После имени архива можно указать конкретные файлы, которые нужно извлечь.

```bash
./build/bin/hamarc -x -f archive.haf file1.txt
```

Можно указать несколько файлов:

```bash
./build/bin/hamarc -x -f archive.haf file1.txt file2.txt
```

В этом случае будут извлечены только перечисленные файлы.

## Добавление файлов в архив

Чтобы добавить файлы в уже существующий архив, используется команда `-a` или `--append`.

```bash
./build/bin/hamarc -a -f archive.haf new_file.txt
```

Можно добавить несколько файлов:

```bash
./build/bin/hamarc -a -f archive.haf new_file1.txt new_file2.txt
```

После добавления файлы появятся в списке содержимого архива:

```bash
./build/bin/hamarc -l -f archive.haf
```

## Удаление файлов из архива

Чтобы удалить файл из архива, используется команда `-d` или `--delete`.

```bash
./build/bin/hamarc -d -f archive.haf file1.txt
```

Можно удалить несколько файлов:

```bash
./build/bin/hamarc -d -f archive.haf file1.txt file2.txt
```

После удаления файлы больше не будут отображаться в списке архива.

## Объединение архивов

Чтобы объединить два архива в один, используется команда `-A` или `--concatenate`.

```bash
./build/bin/hamarc -A -f merged.haf first.haf second.haf
```

В результате будет создан архив `merged.haf`, содержащий файлы из архивов `first.haf` и `second.haf`.

Проверить содержимое объединённого архива можно так:

```bash
./build/bin/hamarc -l -f merged.haf
```

## Пример полного сценария

Создадим несколько файлов:

```bash
echo "hello" > a.txt
echo "world" > b.txt
echo "new file" > c.txt
```

Создадим архив:

```bash
./build/bin/hamarc -c -f test.haf a.txt b.txt
```

Посмотрим содержимое:

```bash
./build/bin/hamarc -l -f test.haf
```

Добавим новый файл:

```bash
./build/bin/hamarc -a -f test.haf c.txt
```

Удалим файл из архива:

```bash
./build/bin/hamarc -d -f test.haf b.txt
```

Извлечём файлы:

```bash
./build/bin/hamarc -x -f test.haf
```

## Особенности реализации

Архиватор использует собственный бинарный формат архива.
В архиве хранится информация о количестве файлов, именах файлов, размерах исходных файлов и закодированные данные.

Данные файлов кодируются с помощью кода Хэмминга.
Это позволяет обнаруживать и исправлять одиночные битовые ошибки при чтении данных из архива.

Архиватор не выполняет сжатие файлов. Его основная задача — хранение нескольких файлов в одном архиве с дополнительной защитой данных.

## Требования

* CMake;
* компилятор C++ с поддержкой стандарта C++20 или новее.
